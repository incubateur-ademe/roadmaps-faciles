---
title: Approbation des contributions
description: Gérez le flux d'approbation des contributions soumises.
---

Lorsque l'option **Modération préalable** est activée dans les [paramètres de l'espace](/doc/admin/general-settings), toutes les nouvelles contributions sont créées avec le statut **En attente** et doivent être approuvées avant d'apparaître sur le tableau.

## Fonctionnement du workflow

```
Nouvelle contribution
      │
      ▼
  En attente ──────► Approuvée (visible sur le tableau)
      │
      └──────────► Rejetée (masquée, consultable dans l'onglet "Rejetés")
```

Les contributions soumises **avant l'activation** de la modération restent visibles — elles ne sont pas rétroactivement placées en attente.

## Interface de modération

Accédez à la section **Modération** depuis le menu de l'espace (visible uniquement pour les modérateurs, administrateurs et le propriétaire).

### Contributions en attente

La liste des contributions soumises et en attente de validation. Pour chaque contribution, vous pouvez :

- **Approuver** — La contribution devient visible sur le tableau.
- **Rejeter** — La contribution est masquée du tableau public. Elle reste accessible dans l'onglet "Rejetés".
- **Supprimer** — La contribution est définitivement supprimée.

Cliquez sur le titre d'une contribution pour **prévisualiser** son contenu complet avant de prendre votre décision.

### Contributions rejetées

La liste des contributions précédemment rejetées. Depuis cet onglet, vous pouvez :

- **Supprimer** définitivement une contribution rejetée.

<Callout type="info">
  Le rejet est une action définitive : une contribution rejetée ne peut pas être ré-approuvée. Si vous souhaitez conserver la possibilité de publier une contribution, préférez la laisser en attente plutôt que de la rejeter.
</Callout>

## Bonnes pratiques

- **Préférez l'attente au rejet** — Une contribution en attente reste disponible pour une approbation future. Le rejet est irréversible (hors suppression + re-soumission). La suppression est également irréversible.
- **Vérifiez le contenu et le ton** avant approbation — assurez-vous que la contribution est pertinente et respectueuse.
- **Communiquez les règles** à vos utilisateurs — Un message d'accueil ou une description de tableau peut expliquer les critères de publication, ce qui réduit le nombre de rejets.
- **Soyez réactif** — Les contributions en attente créent de la frustration chez les utilisateurs. Essayez de traiter la file d'attente régulièrement.
