---
title: Modération du contenu
description: Actions de modération disponibles sur les posts publiés.
---

En dehors du flux d'approbation, les modérateurs et administrateurs disposent d'actions de modération sur les posts déjà publiés.

## Actions disponibles

### Modifier un post

Les modérateurs et administrateurs peuvent modifier le titre et la description de n'importe quel post, même sans l'option "Autoriser les modifications" activée pour les utilisateurs.

Les modifications sont tracées dans la timeline d'activité du post.

### Supprimer un post

Les modérateurs et administrateurs peuvent supprimer n'importe quel post. Cette action est irréversible.

### Changer le statut

Seuls les **administrateurs** et le **propriétaire** peuvent changer le statut d'un post (ex. : passer de "À étudier" à "En cours"). Les modérateurs ne disposent pas de cette permission.

## Journal d'audit

Toutes les actions de modération sont enregistrées dans le **journal d'audit** accessible par les administrateurs. Chaque entrée contient :

- L'action effectuée
- L'utilisateur qui l'a réalisée
- La date et l'heure
- Le résultat (succès ou erreur)
