## Créer un compte

Kokatsuna propose deux méthodes d'authentification :

- **Espace Membre (beta.gouv.fr)** — Si vous êtes membre de l'incubateur, connectez-vous avec votre identifiant Espace Membre.
- **Email** — Entrez votre adresse email et recevez un lien de connexion (magic link). Aucun mot de passe requis.

## Créer un espace de travail

Une fois connecté, rendez-vous dans **Mes espaces** pour créer votre premier espace de travail.

1. Cliquez sur **Créer un espace**.
2. Donnez un **nom** à votre espace (ex. : "Mon Produit").
3. Choisissez un **sous-domaine** unique (ex. : `mon-produit`). Ce sera votre URL : `mon-produit.roadmaps-faciles.fr`.
4. Validez la création.

Votre espace est prêt ! Vous êtes automatiquement **propriétaire** de cet espace.

## Premiers pas

Une fois votre espace créé :

1. **Créez un board** — Rendez-vous dans l'administration pour créer votre premier board (ex. : "Suggestions", "Bugs", "Améliorations").
2. **Configurez les statuts** — Définissez les étapes de votre workflow (ex. : "Planifié", "En cours", "Terminé").
3. **Invitez votre équipe** — Ajoutez des membres avec les rôles appropriés.
4. **Partagez l'URL** — Communiquez l'adresse de votre espace à vos utilisateurs pour qu'ils commencent à soumettre du feedback.

<Callout type="info">
  Vous pouvez également initialiser votre espace avec des données par défaut (boards et statuts prédéfinis) depuis les paramètres généraux.
</Callout>
