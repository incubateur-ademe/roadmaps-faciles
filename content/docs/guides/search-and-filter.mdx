## Barre de recherche

Chaque tableau dispose d'une **barre de recherche** en haut de page. Elle permet de trouver des contributions par mots-clés dans le titre et la description.

La recherche utilise un moteur **full-text** : les résultats prennent en compte les variantes grammaticales et la pertinence des termes.

Pour rechercher :

1. Cliquez sur la barre de recherche ou sur l'icône loupe.
2. Tapez un ou plusieurs mots-clés.
3. Les résultats se mettent à jour automatiquement.

## Filtrer par statut

Utilisez les **badges de statut** visibles sur chaque contribution pour identifier rapidement les idées planifiées, en cours ou terminées. Les statuts sont codés par couleur pour un repérage visuel immédiat.

## Trier les contributions

Les options de tri sont accessibles via les boutons radio en haut du tableau :

| Tri | Description |
|-----|-------------|
| **Tendances** | Contributions les plus actives récemment (combinaison de votes et commentaires récents). Tri par défaut. |
| **Top** | Contributions avec le plus de votes, tous temps confondus. |
| **Récents** | Contributions les plus récemment créées. |

## Filtrer par tags

Les **tags** (étiquettes) associés aux contributions permettent un filtrage thématique. Les tags sont visibles sous le titre de chaque contribution, en vue cartes comme en vue liste.
