title: Soumettre un post
description: Comment créer et gérer vos propositions.
---

## Créer un post

1. Rendez-vous sur un **board** de l'espace de travail.
2. Cliquez sur le bouton **Soumettre une idée** (ou équivalent).
3. Remplissez le formulaire :
   - **Titre** — Un résumé clair de votre proposition (minimum 3 caractères).
   - **Description** — Des détails complémentaires. Le format **Markdown** est supporté pour la mise en forme.
4. Si disponible, cochez l'option **Soumettre anonymement** pour masquer votre identité.
5. Validez votre soumission.

<Callout type="info">
  Si la modération est activée sur l'espace, votre post sera en attente de validation avant d'apparaître publiquement.
</Callout>

## Modifier un post

Si l'option est activée par l'administrateur, vous pouvez modifier vos propres posts après leur création :

1. Ouvrez le post que vous souhaitez modifier.
2. Cliquez sur le bouton de modification.
3. Mettez à jour le titre ou la description.
4. Enregistrez vos modifications.

L'historique des modifications est conservé et visible dans la timeline du post.

## Supprimer un post

Si l'option est activée, vous pouvez supprimer vos propres posts. Cette action est irréversible.
