## Créer une contribution

Le formulaire de soumission est visible directement sur chaque tableau, dans la colonne de gauche :

1. Rendez-vous sur un **tableau** de l'espace de travail.
2. Remplissez le formulaire **Soumettre une suggestion** :
   - **Titre** — Un résumé clair de votre proposition (minimum 3 caractères).
   - **Description** — Des détails complémentaires, via l'**éditeur Markdown** intégré :
     - **Barre d'outils** — Gras, italique, titres, listes, citations, code, liens.
     - **Raccourcis clavier** — `Ctrl+B` (gras), `Ctrl+I` (italique).
     - **Images** — Glissez-déposez une image, collez depuis le presse-papier, ou utilisez le bouton image dans la barre d'outils. L'image est uploadée automatiquement et insérée dans le texte.
     - **Aperçu** — Activez le toggle "Aperçu" dans la barre d'outils pour prévisualiser le rendu final.
3. Cliquez sur **Valider**.

Votre contribution apparaît immédiatement sur le tableau (ou en attente de modération si cette option est activée).

### Soumission anonyme

Si le **feedback anonyme** est activé sur l'espace, vous pouvez soumettre une idée sans être connecté. Votre contribution apparaîtra avec l'auteur "Anonyme".

<Callout type="info">
  Si la modération préalable est activée, votre contribution sera en **attente de validation** par un modérateur avant d'apparaître publiquement.
</Callout>

## Modifier une contribution

Si l'option **Autoriser les modifications** est activée par l'administrateur, vous pouvez modifier vos propres contributions après publication :

1. Ouvrez la page de détail de votre contribution.
2. Cliquez sur le bouton de modification.
3. Mettez à jour le titre ou la description.
4. Enregistrez vos modifications.

La mention "modifié par" apparaît sur la page de la contribution avec la date de dernière modification.

## Supprimer une contribution

Si l'option **Autoriser la suppression** est activée, vous pouvez supprimer vos propres contributions.

<Callout type="warn">
  La suppression est irréversible. Les votes et commentaires associés sont également supprimés.
</Callout>
