## Accéder à son profil

Votre profil est accessible depuis le **menu utilisateur** (en haut à droite de l'interface), puis en cliquant sur **Profil** ou **Mon compte**.

<ImageWithTheme srcLight="/doc/screenshots/profile-light.png" srcDark="/doc/screenshots/profile-dark.png" alt="Page de profil" />

## Informations modifiables

### Nom

Vous pouvez modifier le nom qui s'affiche sur vos contributions et commentaires.

### Email

L'adresse email associée à votre compte est celle utilisée pour la connexion par magic link. Vous pouvez la mettre à jour depuis votre profil.

### Préférences de notification

Activez ou désactivez les notifications par email pour rester informé des évolutions sur les contributions que vous suivez.

## Sécurité et double authentification

Depuis **Profil > Sécurité**, vous pouvez configurer une ou plusieurs méthodes de double authentification (2FA) :

- **Clé d'accès (Passkey)** — Authentification biométrique ou clé de sécurité matérielle.
- **Application OTP** — Code temporaire via une application (Google Authenticator, Authy, etc.).
- **Email** — Code de vérification envoyé par email.

Si l'administrateur de votre espace a activé la 2FA obligatoire, vous devrez configurer au moins une méthode pour continuer à accéder à l'espace.

## Liaison avec l'Espace Membre

Si vous êtes membre de beta.gouv.fr, vous pouvez **lier votre compte Roadmaps Faciles à votre compte Espace Membre**. Cette liaison permet de :

- Synchroniser votre adresse email
- Confirmer votre statut de membre beta.gouv.fr

Vous pouvez aussi **délier** votre compte Espace Membre à tout moment.

## Supprimer son compte

La suppression de votre compte est irréversible. Elle entraîne :

- L'anonymisation de vos contributions et commentaires (ils restent visibles mais l'auteur passe à "Utilisateur supprimé").
- La suppression de vos memberships sur tous les espaces de travail.
- La suppression de vos sessions et clés API.

Vos votes sont conservés mais anonymisés — ils continuent de compter dans les totaux.

<Callout type="warn">
  Cette action est définitive. Vos contributions ne pourront plus être rattachées à votre identité.
</Callout>
