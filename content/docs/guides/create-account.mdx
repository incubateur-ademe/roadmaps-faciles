## Méthodes de connexion

Roadmaps Faciles propose plusieurs méthodes d'authentification selon votre contexte.

### Connexion par email (magic link)

C'est la méthode la plus courante pour se connecter à un espace de travail :

1. Rendez-vous sur la page de connexion de l'espace.
2. Entrez votre **adresse email**.
3. Cliquez sur **Se connecter**.
4. Consultez votre boîte email et cliquez sur le **lien de connexion** reçu.

Le lien est valable pour une seule utilisation. Aucun mot de passe n'est requis — c'est ce qu'on appelle un "magic link".

### Connexion via Espace Membre (beta.gouv.fr)

Si vous êtes membre de l'écosystème beta.gouv.fr :

1. Rendez-vous sur la page d'accueil de Roadmaps Faciles (domaine racine).
2. Entrez votre **identifiant Espace Membre** (ex. : `prenom.nom`).
3. Cliquez sur **Se connecter**.
4. Consultez votre email et cliquez sur le lien reçu.

Une fois connecté sur la page d'accueil, vous pouvez accéder à n'importe quel espace de travail grâce au **pont SSO** : votre session est automatiquement transférée lorsque vous naviguez vers un sous-domaine.

### Connexion via invitation

Si vous recevez un email d'invitation :

1. Cliquez sur le **lien d'invitation** dans l'email.
2. Vous arrivez sur la page de connexion de l'espace avec l'invitation pré-chargée.
3. Connectez-vous normalement (email ou Espace Membre).
4. Votre compte est automatiquement rattaché à l'espace avec le rôle défini par l'administrateur.

## Inscription et politique d'accès

Chaque espace de travail définit sa propre politique d'inscription :

- **Tout le monde** — N'importe quelle adresse email peut créer un compte.
- **Domaines autorisés** — Seules les adresses email d'un domaine spécifique sont acceptées (ex. : `@mon-organisation.fr`).
- **Sur invitation uniquement** — L'inscription est fermée, seuls les membres invités peuvent rejoindre l'espace.

Si vous n'arrivez pas à vous inscrire, vérifiez la politique d'accès avec l'administrateur de l'espace.

## Appartenir à plusieurs espaces

Un même compte (identifié par votre email) peut être membre de **plusieurs espaces de travail**. Vous pouvez avoir des rôles différents sur chaque espace.
