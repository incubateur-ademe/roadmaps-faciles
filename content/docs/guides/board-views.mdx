title: Vues des tableaux
description: Basculez entre vue cartes et vue liste compacte.
---

Roadmaps Faciles propose deux modes d'affichage pour les contributions d'un tableau. Les boutons de sélection sont situés en haut à droite du tableau.

## Vue cartes

La **vue cartes** est le mode par défaut. Chaque contribution est affichée sous forme de carte avec :

- Le compteur de votes (cliquable)
- Le titre (lien vers la page de détail)
- Le statut (badge coloré)
- Les tags
- L'auteur et le nombre de commentaires

<ImageWithTheme srcLight="/doc/screenshots/board-cards-light.png" srcDark="/doc/screenshots/board-cards-dark.png" alt="Vue cartes" />

Ce mode est adapté à la **consultation détaillée** : on peut évaluer chaque contribution d'un coup d'oeil.

## Vue liste compacte

La **vue liste** affiche les contributions sous forme de lignes condensées, avec :

- Le compteur de votes
- Le statut
- Le titre
- L'auteur, la date relative et les tags
- Le nombre de commentaires

<ImageWithTheme srcLight="/doc/screenshots/board-list-light.png" srcDark="/doc/screenshots/board-list-dark.png" alt="Vue liste" />

Ce mode est adapté au **parcours rapide** d'un grand nombre de contributions. Il est inspiré de la vue GitHub Issues.

## Basculer entre les vues

Cliquez sur les boutons **Vue cartes** ou **Vue liste** en haut du tableau. Le choix est conservé dans l'URL (`?view=list` ou `?view=cards`) — vous pouvez donc partager un lien vers une vue spécifique.

Le tri (tendances, top, récents) fonctionne de la même façon dans les deux vues.
