## Voter pour une contribution

Le vote est le mécanisme principal de priorisation dans Roadmaps Faciles. Plus une contribution reçoit de votes, plus elle remonte dans le classement.

- Cliquez sur le **compteur de votes** à côté d'une contribution (le nombre s'incrémente).
- Chaque utilisateur ne peut voter **qu'une seule fois** par contribution.
- Le vote peut être **retiré** en cliquant à nouveau sur le compteur.

Le compteur est visible aussi bien en vue cartes qu'en vue liste.

### Vote anonyme

Si le **vote anonyme** est activé par l'administrateur, les visiteurs non-connectés peuvent également voter. Leur vote est identifié par un cookie de navigateur.

## Commenter une contribution

Si les commentaires sont activés sur l'espace :

1. Ouvrez une contribution pour afficher sa **page de détail**.
2. Rédigez votre commentaire dans le champ **Ajouter un commentaire** — l'éditeur Markdown intégré est disponible (mise en forme, images, aperçu).
3. Validez pour publier.

Les commentaires apparaissent dans la **timeline d'activité** de la contribution, aux côtés des changements de statut et des votes agrégés. Cette timeline offre une vision chronologique complète de la vie d'une contribution.

### Répondre à un commentaire

Cliquez sur le bouton **Répondre** sous un commentaire pour ouvrir un champ de réponse. Les réponses sont affichées en fil de discussion sous le commentaire parent.

### Modifier ou supprimer un commentaire

- **Modifier** — Cliquez sur l'icône de modification à côté de votre commentaire pour l'éditer. La mention "(modifié)" apparaît après enregistrement.
- **Supprimer** — Cliquez sur l'icône de suppression. Une confirmation est demandée avant la suppression définitive.

Les administrateurs et modérateurs peuvent supprimer n'importe quel commentaire.

<Callout type="warn">
  La suppression d'un commentaire est irréversible. Si le commentaire a des réponses, celles-ci sont également supprimées.
</Callout>

## Suivre une contribution

Vous pouvez **suivre** une contribution pour être informé de son évolution (changements de statut, nouveaux commentaires). Le suivi est disponible depuis la page de détail de la contribution.
