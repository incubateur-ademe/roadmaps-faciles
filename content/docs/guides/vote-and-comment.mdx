## Voter pour un post

Le vote est le mécanisme principal de priorisation dans Kokatsuna. Plus un post reçoit de votes, plus il remonte dans le classement.

- Cliquez sur le bouton de **vote** (icône pouce/cœur) à côté d'un post.
- Chaque utilisateur ne peut voter **qu'une seule fois** par post.
- Le vote peut être retiré en cliquant à nouveau sur le bouton.

<Callout type="info">
  Si le vote anonyme est activé, les visiteurs non-connectés peuvent également voter.
</Callout>

## Commenter un post

Si les commentaires sont activés sur l'espace :

1. Ouvrez un post pour afficher sa page de détail.
2. Rédigez votre commentaire dans le champ prévu.
3. Publiez votre commentaire.

Les commentaires apparaissent dans la **timeline d'activité** du post, aux côtés des changements de statut et des votes.

## Trier les posts

Utilisez les options de tri pour explorer les posts d'un board :

- **Tendances** — Les posts les plus actifs récemment.
- **Top** — Les posts avec le plus de votes.
- **Récents** — Les derniers posts créés.

## Rechercher

Utilisez la barre de recherche pour trouver des posts par mots-clés dans le titre ou la description.
