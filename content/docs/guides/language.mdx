title: Changer de langue
description: Basculez l'interface entre français et anglais.
---

Roadmaps Faciles est disponible en **français** (par défaut) et en **anglais**.

## Changer la langue de l'interface

Le sélecteur de langue se trouve dans le **pied de page** de chaque page :

1. Faites défiler jusqu'au bas de la page.
2. Cliquez sur le bouton de sélection de langue.
3. Choisissez **Français** ou **English**.
4. La page se recharge automatiquement dans la langue choisie.

## Persistance du choix

Votre préférence de langue est stockée dans un **cookie** de navigateur. Elle est conservée entre les sessions : vous n'avez pas besoin de re-sélectionner la langue à chaque visite.

## Contenu concerné

Le changement de langue s'applique à :

- L'interface (menus, boutons, messages système)
- Les messages de validation (erreurs de formulaire)
- Les dates et formats numériques

Le **contenu utilisateur** (titres et descriptions des contributions, commentaires, noms des tableaux et statuts) n'est pas traduit — il reste dans la langue dans laquelle il a été rédigé.
