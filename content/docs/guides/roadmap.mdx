---

La **roadmap** offre une vue d'ensemble de l'avancement des posts sous forme de tableau Kanban.

## Comment ça fonctionne

- Chaque **colonne** représente un statut (ex. : "Planifié", "En cours", "Terminé").
- Les posts sont affichés dans la colonne correspondant à leur statut actuel.
- Seuls les statuts marqués comme **visibles dans la roadmap** par l'administrateur y apparaissent.

## Navigation

- Cliquez sur le titre d'un post pour accéder à sa page de détail.
- Le nombre de votes est affiché sur chaque carte.
- Le board d'origine est indiqué par un badge de couleur.

## Accès

La roadmap est accessible depuis le menu principal si l'option **Afficher la roadmap dans l'en-tête** est activée par l'administrateur. Sinon, elle reste accessible via l'URL directe `/roadmap`.
