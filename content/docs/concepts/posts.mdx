title: Posts

Un **post** est une contribution soumise par un utilisateur sur un board. Il représente une idée, une suggestion, un bug ou toute autre forme de retour.

## Contenu d'un post

- **Titre** — Résumé court de la proposition (minimum 3 caractères).
- **Description** — Détails de la proposition (Markdown supporté, optionnel).
- **Board** — Le board auquel le post est rattaché.
- **Statut** — L'état d'avancement attribué par un administrateur.
- **Tags** — Étiquettes personnalisées pour catégoriser le post.

## Interactions

### Votes

Les utilisateurs peuvent voter pour les posts qui les intéressent. Chaque utilisateur ne peut voter qu'une seule fois par post. Le vote est un mécanisme de priorisation : plus un post a de votes, plus il est considéré comme important.

### Commentaires

Si les commentaires sont activés, les utilisateurs peuvent discuter autour d'un post. Les commentaires apparaissent dans la timeline d'activité du post.

### Timeline d'activité

Chaque post dispose d'une timeline qui retrace :
- Les changements de statut
- Les commentaires ajoutés
- Les votes agrégés

## Posts anonymes

Si l'espace autorise le feedback anonyme, un post peut être soumis sans compte utilisateur. L'auteur apparaît alors comme "Anonyme".

## Approbation

Si la **modération préalable** est activée sur l'espace, les nouveaux posts sont créés avec le statut "En attente" et doivent être approuvés par un modérateur avant d'apparaître publiquement sur le board.
