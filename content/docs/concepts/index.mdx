Roadmaps Faciles s'articule autour de quelques concepts clés. Cette section vous aide à comprendre comment ils s'imbriquent.

```
Espace de travail
├── Tableaux (thèmes de feedback)
│   ├── Contributions (suggestions, idées, bugs)
│   │   ├── Votes
│   │   ├── Commentaires
│   │   └── Statut (planifié, en cours, terminé...)
│   └── ...
├── Feuille de route (vue kanban des statuts)
├── Membres (propriétaire, admin, modérateur, utilisateur)
└── Paramètres (visibilité, modération, domaine...)
```

## Les concepts en bref

| Concept | Description |
|---------|-------------|
| [**Espace de travail**](/doc/concepts/workspaces) | Un environnement isolé pour votre produit ou équipe, accessible via un sous-domaine dédié. |
| [**Tableau**](/doc/concepts/boards) | Un espace thématique qui regroupe des contributions autour d'un sujet (suggestions, bugs, etc.). |
| [**Contribution**](/doc/concepts/posts) | Une idée, suggestion ou remontée soumise par un utilisateur sur un tableau. |
| [**Statut**](/doc/concepts/statuses) | L'état d'avancement d'une contribution dans votre workflow (planifié, en cours, terminé...). |
| [**Rôle**](/doc/concepts/roles) | Le niveau de permission d'un membre dans un espace (propriétaire, admin, modérateur, utilisateur). |

Chaque concept est détaillé dans les pages suivantes. Pour un rappel rapide des termes, consultez le [glossaire](/doc/concepts/glossary).
