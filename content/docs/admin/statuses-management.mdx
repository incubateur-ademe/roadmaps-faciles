title: Gestion des statuts
description: Définissez votre workflow avec des statuts personnalisés.
---

## Créer un statut

1. Rendez-vous dans **Administration > Statuts**.
2. Cliquez sur **Créer un statut**.
3. Configurez :
   - **Nom** — Le libellé du statut (ex. : "En cours de développement").
   - **Couleur** — Choisissez parmi les couleurs du système de design DSFR.
   - **Visible dans la roadmap** — Cochez si ce statut doit apparaître dans la vue roadmap.
4. Validez.

## Organiser les statuts

L'ordre des statuts reflète la progression naturelle de votre workflow. Réordonnez-les pour que la roadmap affiche les colonnes dans l'ordre logique.

## Modifier un statut

Cliquez sur un statut pour modifier ses propriétés.

## Supprimer un statut

La suppression d'un statut retire le statut de tous les posts qui l'utilisaient. Les posts restent visibles mais sans statut assigné.
