## Politique d'inscription

Contrôlez qui peut s'inscrire sur votre espace de travail :

| Politique | Description |
|-----------|-------------|
| **Tout le monde** | Toute personne peut créer un compte et rejoindre l'espace. |
| **Personne** | L'inscription est fermée. Seules les invitations sont possibles. |
| **Domaines autorisés** | Seuls les emails correspondant aux domaines configurés sont acceptés. |

### Domaines autorisés

Lorsque la politique est définie sur **Domaines autorisés**, vous pouvez gérer la liste des domaines email acceptés.

Par exemple, ajouter `mon-organisation.fr` autorisera toutes les adresses `*@mon-organisation.fr` à s'inscrire.

1. Ajoutez un domaine dans le champ dédié.
2. Validez.
3. Supprimez un domaine en cliquant sur le bouton de suppression correspondant.
