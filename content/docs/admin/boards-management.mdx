---
title: Gestion des boards
description: Créez, organisez et configurez vos boards de feedback.
---

## Créer un board

1. Rendez-vous dans **Administration > Boards**.
2. Cliquez sur **Créer un board**.
3. Renseignez :
   - **Nom** — Le titre du board (ex. : "Suggestions").
   - **Description** — Une description optionnelle (Markdown supporté).
4. Validez la création.

## Organiser les boards

Les boards sont affichés dans l'ordre défini dans l'administration. Vous pouvez :

- **Réordonner** les boards en les déplaçant vers le haut ou vers le bas.
- **Définir un board principal** — Ce board sera affiché comme page d'accueil de l'espace.

## Modifier un board

Cliquez sur un board pour modifier son nom ou sa description.

## Supprimer un board

<Callout type="warn">
  La suppression d'un board entraîne la suppression de tous les posts qu'il contient. Cette action est irréversible.
</Callout>
