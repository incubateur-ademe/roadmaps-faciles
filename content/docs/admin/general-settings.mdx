---
title: Paramètres généraux
description: Configurez les options principales de votre espace de travail.
---

Les paramètres généraux contrôlent le comportement global de votre espace de travail.

## Identité

- **Nom de l'espace** — Le nom affiché dans l'en-tête et les métadonnées.
- **Sous-domaine** — L'identifiant unique utilisé dans l'URL (non modifiable après création).
- **Domaine personnalisé** — Associez votre propre nom de domaine.
- **Langue** — Langue par défaut de l'interface (français ou anglais).

## Visibilité et accès

| Option | Description |
|--------|-------------|
| **Espace privé** | Seuls les membres invités peuvent accéder au contenu. |
| **Feedback anonyme** | Les visiteurs non-connectés peuvent soumettre des posts. |

## Options de feedback

| Option | Description | Défaut |
|--------|-------------|--------|
| **Votes** | Activer/désactiver le système de vote. | Activé |
| **Commentaires** | Activer/désactiver les commentaires sur les posts. | Activé |
| **Vote anonyme** | Les visiteurs non-connectés peuvent voter. | Activé |
| **Modification des posts** | Les auteurs peuvent modifier leurs posts après publication. | Désactivé |
| **Suppression des posts** | Les auteurs peuvent supprimer leurs propres posts. | Désactivé |
| **Modération préalable** | Les posts doivent être approuvés par un modérateur. | Désactivé |

## Navigation

- **Roadmap dans l'en-tête** — Affiche un lien vers la roadmap dans le menu principal.

## Données initiales

Pour un espace vide, vous pouvez initialiser des boards et statuts par défaut en un clic.

## Zone de danger

<Callout type="warn">
  Les actions suivantes sont irréversibles.
</Callout>

- **Purger les données** — Supprime tous les posts, votes et commentaires de l'espace. Les boards, statuts et membres sont conservés.
- **Supprimer l'espace** — Supprime définitivement l'espace et toutes ses données.
