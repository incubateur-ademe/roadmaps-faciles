---
title: Clés API
description: Gérez l'accès programmatique à votre espace de travail.
---

Les **clés API** permettent un accès programmatique à votre espace de travail pour des intégrations ou des automatisations.

## Créer une clé API

1. Rendez-vous dans **Administration > API**.
2. Cliquez sur **Créer une clé API**.
3. Donnez un nom descriptif à votre clé.
4. Validez la création.

<Callout type="warn">
  La clé API n'est affichée qu'une seule fois à la création. Copiez-la immédiatement et conservez-la en lieu sûr.
</Callout>

## Gérer les clés

La liste affiche vos clés existantes avec :
- Le nom de la clé
- La date de création
- Un bouton de suppression

## Supprimer une clé

La suppression d'une clé est immédiate et irréversible. Toute intégration utilisant cette clé cessera de fonctionner.
